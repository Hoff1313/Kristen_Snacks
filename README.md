# Disney Snack Tracker

A GitHub Pages-ready mini site built from your Disney snack notes.

## What it includes
- Snack cards with images
- Park/area filtering
- Sorting by park, name, or tried status
- Checkboxes that save in the browser using localStorage
- Easy editing in `data.js`

## Publish to GitHub Pages
1. Create a new GitHub repository.
2. Upload everything in this folder.
3. In the repository, open **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/(root)** folder, then save.
6. GitHub will give you a public Pages URL for your snack tracker.

## Customize later
- Update snack names, locations, or park tags in `data.js`.
- Replace any placeholder images in `images/` with your own photos and keep the same filenames, or update the image paths in `data.js`.
- Adjust colors and layout in `styles.css`.

## Notes
Some park assignments are directly stated in the notes. A few are inferred from venue names so the list can sort cleanly by area.
